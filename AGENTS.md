# AI エージェント用基本方針

## 目的
本ドキュメントは、AI エージェントが本プロジェクトで作業を行う際の共通の作業方針を定義します。

## 基本方針
- **会話言語**: 日本語
- **コード内コメント**: 日本語
- **エラーメッセージ**: 英語
- **コミットメッセージ**: Conventional Commits ( `<description>` は日本語)
- **日本語と英数字の間**: 半角スペースを挿入

## 判断記録のルール
判断を行った際は、以下の内容を明示するように努めてください。
1. 判断内容の要約
2. 検討した代替案
3. 採用理由と、採用しなかった案の理由
4. 前提条件・仮定・不確実性

## 開発手順
1. **プロジェクト理解**: 既存のコード、ディレクトリ構造、`package.json` を確認する。
2. **依存関係インストール**: `pnpm install` を実行する。
3. **実装**: プロジェクトの規約 (TypeScript, ESLint, Prettier) に従い変更を行う。
4. **検証**: `pnpm lint` および `pnpm test` を実行し、問題がないことを確認する。

## セキュリティ / 機密情報
- API キー、トークン、パスワード等を Git にコミットしない。
- 環境変数または `data/config.json` を使用する。
- ログに機密情報を出力しない。

## リポジトリ固有
- このプロジェクトは Discord Bot です。
- 翻訳には Google Apps Script を使用します。
- メッセージがクロスポストされた際に動作するロジックが中心です。
